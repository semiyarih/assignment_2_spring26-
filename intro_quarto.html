Introduction to Quarto

What Is Quarto?

Quarto is a modern publishing system that lets you combine text, code, and output in one document.
You can create reports, slides, websites, books, and more — all from RStudio.

This tutorial walks you through the basics so you can start using Quarto right away.

1. Creating a New Quarto Document

  1. Open RStudio
  2. Go to: File > New File > Quarto Document
  3. Choose:
    • Title
    • Author
    • Format
  4. Click Create

You’ll see a new .qmd file with a header (YAML) and some starter content.

2. Understanding the YAML Header

The YAML block at the top controls the document settings:

--- 
title: "My First Quarto Document" 
author: "Student Name" 
format: html 

---

3. Writing Text

Quarto uses Markdown, which is simple and intuitive.

Examples:

Heading 1

Heading 2

Heading 3

This is bold, italic, and inline code.

  • Bullet list item
  • Another itm
  1. Numbered item
  2. Another item

4. Adding R Code Chunks

Use triple backticks with {r}:

# A simple calculation
x <- 1:10
mean(x)
[1] 5.5

Or a plot

plot(cars)

When you render the document, the code runs and the output appears below it.

5. Chunk Options

Control how code behaves:

     speed           dist       
 Min.   : 4.0   Min.   :  2.00  
 1st Qu.:12.0   1st Qu.: 26.00  
 Median :15.0   Median : 36.00  
 Mean   :15.4   Mean   : 42.98  
 3rd Qu.:19.0   3rd Qu.: 56.00  
 Max.   :25.0   Max.   :120.00

Common options:

  • echo: false hide code

  • message: false hide messages

  • warning: false hide warnings

  • eval: false This option will not excute the chunk of code

  • fig-width / fig-height control plot size

8. Using Citations (Optional)

9. Rendering the Document

Click the Render button at the top of RStudio.

This creates an HTML (or PDF/Word) output file.
